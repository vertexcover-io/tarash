# tarash-captions

AI-powered video caption generation for the tarash ecosystem.

## Installation

```bash
uv add tarash-captions
```

## Usage

```python
from tarash.captions import CaptionGenerator
```

## Development

```bash
# Install in development mode
uv sync

# Run tests
uv run pytest

# Type checking
uv run pyright
```
